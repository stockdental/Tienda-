# 🦷 StockDental | Expert Division v3.0
**Plataforma Digital de Suministros Dentales - Maracay, Venezuela**

## 🚀 Visión General
StockDental es una solución web de alto rendimiento diseñada específicamente para la distribución técnica de insumos odontológicos y de laboratorio. Orientada a profesionales que exigen precisión, la plataforma combina una interfaz minimalista (inspirada en estándares de usabilidad de aplicaciones de salud) con una arquitectura ligera y funcional.

## 🛠️ Stack Tecnológico
*   **Frontend:** HTML5 Semántico y CSS3.
*   **Framework de Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first framework) para garantizar un diseño responsivo y moderno.
*   **Motor de Lógica:** JavaScript (Vanilla ES6+) para la gestión de estados del carrito de compras y cálculos financieros.
*   **Despliegue:** GitHub Pages para una disponibilidad del 99.9% sin costes de infraestructura.

## 📊 Especificaciones Técnicas de Negocio
La plataforma opera bajo una lógica de protección de márgenes operativos:
- **Tasa de Cambio:** Sincronizada con indicadores de mercado (Binance/BCV).
- **Márgenes de Utilidad:** Parametrizados entre el 10% y el 25% según la categoría del insumo.
- **Métodos de Liquidación:** Integración directa con Pago Móvil (Mercantil) y pasarelas digitales (Binance).

## 🦷 Líneas de Producto Iniciales
1. **Laboratorio:** Yesos de alta precisión (Whip Mix T4).
2. **Bioseguridad:** Protección de grado clínico (Guantes Ecomed).
3. **Prótesis:** Ceras base y modelado (Bon Dent).
4. **Ortodoncia:** Insumos para zocalado y estudio.
5. **Resinas:** Acrílicos de alta pureza monomérica.
